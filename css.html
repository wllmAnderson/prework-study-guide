CSS Study Guide

1. Introduction:

  • Definition: CSS (Cascading Style Sheets) is a stylesheet language used to describe the look and formatting of a document written in HTML.
  • Purpose: To separate content from presentation and enable web developers to control layout, colors, fonts, and more.

2. Basics of CSS

  • Declaration Block: Contains one or more declarations separated by semicolons.
  • Properties: The attributes you want to change, such as color, font-size, or margin.
  • Values: Each property is assigned a value to specify its style.

Example:

          p {
              color: blue;
              font-size: 16px;
          }

3. Selectors

  • Element Selector: Targets a specific HTML element, e.g., p, h1.
  • Class Selector: Begins with a period, e.g., .classname.
  • ID Selector: Begins with a hash, e.g., #idname.
  • Descendant Selector: Targets an element that sits inside another specific element.
  • Grouping Selector: Combines multiple selectors into one rule set.

4. Box Model

  • Content: The actual content, like text or an image.
  • Padding: Space between content and the border.
  • Border: Defines the boundary of the box.
  • Margin: Outermost layer; space outside the border.

5. Layout

  • Display Property: Determines the display level of an element, e.g., block, inline, none.
  • Positioning: Describes how elements are positioned, e.g., static, relative, fixed, absolute.
  • Flexbox: A layout model that allows items to align and distribute space within a container.
  • CSS Grid: A two-dimensional layout system for the web.

6. Colors and Backgrounds

  • Colors: Can be defined by names (red), HEX (#FF0000), RGB (rgb(255,0,0)), or HSL (hsl(0, 100%, 50%)).
  • Background Properties: Used to define the background effects of an element, e.g., background-color, background-image.

Text and Fonts

  • Text Properties: Control the appearance and spacing of text, e.g., text-align, text-decoration.
  • Font Properties: Control the font family, size, weight, and style.

7. Responsive Design

  • Media Queries: Used to apply styles based on device conditions.
  • Viewport: Used to set the width and scaling of the display area for web content.

8. Advanced Concepts

  • Transitions: Create gradual changes over a property's value.
  • Animations: Create complex animations by gradually changing from one set of styles to another.
  • Transforms: Apply 2D or 3D transformations to an element.

9. Best Practices

  • Commenting: Helps maintain clarity in your code.
  • Organization: Keep related styles grouped, and use a consistent naming convention.
  • Optimization: Minimize the use of heavy graphics, keep code DRY (Don't Repeat Yourself), and use shorthand properties where possible.
