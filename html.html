Introduction to HTML

HTML, or HyperText Markup Language, is the standard markup language for creating Web pages. It provides the structural framework for web content, allowing developers to organize elements coherently. Every web page is fundamentally an HTML document.

Key Components of HTML

  • Elements: Represented by tags such as h1, p, and a.
  • Attributes: Provide additional information about elements. Example: href for links, src for images.
  • Document Object Model (DOM): Represents the structure of an HTML document in a tree-like format.

Structure of an HTML Document

  • Document Declaration (!DOCTYPE): Defines the version of HTML, e.g., HTML5.
  • HTML Element (<html>): The root element of a document.
  • Head Element (<head>): Contains meta-information, links to stylesheets, and scripts.
  • Body Element (<body>): The main content area of a web page.

Inside the Body Element

  • Headers (h1, h2,...h6): Define headings.
  • Paragraphs (p): Contain blocks of text.
  • Links (a): Used for navigation.
  • Lists (ol, ul): Group items together.
  • Images (img): Display graphics.

HTML Attributes

Attributes provide additional information about elements. They are defined using a name-value pair structure in the opening tag of an element. While many attributes can be used universally, some are element-specific.

  • href: Specifies the destination URL for a link.
  • src: Specifies the source path for images.
  • alt: Provides a textual alternative for images.
  • class & id: Important for styling and scripting to identify elements or groups.

Basic Elements

  • <h1> to <h6>: Headings, with <h1> being the highest level.
  • <p>: Paragraph element.
  • <a>: Anchor (hyperlink) element, often used with the "href" attribute.
  • <img>: Image element, commonly used with the "src" attribute.

Lists

  • <ul>: Unordered list.
  • <ol>: Ordered list.
  • <li>: List item, used within both ordered and unordered lists.

Forms

  • <form>: Form element, used to collect user input.
  • <input>: Input field, can vary in type (text, radio, checkbox, etc.).
  • <label>: Label for an <input> element.
  • <textarea>: Multi-line text input.
  • <select> and <option>: Dropdown list.

Table Elements

  • <table>: Defines a table.
  • <tr>: Table row.
  • <td>: Table data/cell.
  • <th>: Table header.

Semantic Elements

  • <header>, <footer>: Header and footer of a section or page.
  • <nav>: Navigation links.
  • <article>: A self-contained composition.
  • <section>: A standalone section of a page.
  • <aside>: Content aside from the main content.

Attributes

  • href: Specifies the URL for a link.
  • src: Specifies the source URL for an embedded content.
  • alt: Alternative text for media elements.
  • class: Specifies one or more class names for an element.
  • id: Specifies a unique id for an element.
